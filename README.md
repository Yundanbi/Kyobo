## 🛠️ 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| Frontend | HTML, CSS, JavaScript, JSP |
| Backend  | Java Servlet, JDBC |
| Database | MySQL |
| IDE | Eclipse |
| ETC | Git, GitHub |

---

## ✨ 주요 기능

- 👤 회원가입 / 로그인 / 로그아웃
- 📝 게시판 등록, 수정, 삭제
- 📚 관리자만 도서등록, 수정, 삭제
